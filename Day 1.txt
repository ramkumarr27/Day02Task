
Question 1 :
What is the difference between http and https...??
Answer :
HTTP -
HTTP stands for hyper text transfer protocol.
It works at the application layer.
The default port number is 80 for communication.
No  Encryption is present in HTTP websites
HTTPS-
HTTPS stands for hyper text transfer protocol secure.
It works at the transport layer.
The default port number is 443 for communication.
Both Encryption and Decryption exists on HTTPS websites
